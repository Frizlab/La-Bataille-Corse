Le jeu

Dans le jeu, les cartes retournées représentent les joueurs.
Un point bleu est présent sur la carte représentant le joueur dont c'est le tour.

Les cartes sont posées en appuyant sur espace.

Pendant le jeu, n'importe qui peut taper à n'importe quel moment. Pour savoir quand il faut taper, voir Quand taper ?.
